В роутерах Keenetic есть возможность добавлять DNS-записи без развёртывания отдельного DNS-сервера.

1. Подключаемся к роутеру.
   #+begin_src console
   $ telnet <ip-адрес роутера>
   <вводим логин и пароль>
   #+end_src

2. Добавляем DNS-запись для домена.
   - Замените =<domain>= на нужный домен.
   - Замените =<ip>= на IP-адрес, который вы хотите связать с доменом.

   #+begin_src console
   $ ip host <domain> <ip>
   $ system configuration save
   #+end_src

3. Для удаления DNS-записи используйте команду =no ip host=.
   #+begin_src console
   $ no ip host <domain>
   $ system configuration save
   #+end_src
